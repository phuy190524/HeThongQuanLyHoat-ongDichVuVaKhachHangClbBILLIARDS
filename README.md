# Hệ thống quản lý CLB Billiards Quốc Vinh

## Mô tả
Dự án phân tích và thiết kế hệ thống quản lý dịch vụ và khách hàng cho CLB Billiards Quốc Vinh.

## Nội dung thực hiện
- Phân tích yêu cầu hệ thống
- Thiết kế kiến trúc hệ thống
- Xây dựng mô hình dữ liệu
- Thiết kế chức năng quản lý khách hàng và dịch vụ

## Chức năng chính
- Quản lý khách hàng
- Quản lý dịch vụ (bàn, giờ chơi)
- Quản lý hoạt động CLB
- Thống kê và báo cáo

## Tài liệu
- File Word: báo cáo phân tích và thiết kế hệ thống
- Sơ đồ hệ thống 

## Công nghệ / Công cụ
- Microsoft Word
- Phân tích hệ thống (UML, mô hình hóa)

## Mục tiêu
Hiểu quy trình phân tích và thiết kế hệ thống thực tế, làm nền tảng cho phát triển phần mềm.
